Home page

You can see two animated buttons "Add Note" and "Add CheckList" on Home page. Click the desirable button to create Note, or CheckList

Add Note

You can see an animated button "Add Note". Click the button to create your note, You can add title and description.

You can find four icons select the required icon:

  • Save - Saves in Notes
  • Reminder -Saves in Notes and Reminders
  • Favourite - Saves in Notes and Favourites
  • Cancel

Add list

You can see an animated button "Add CheckList". Click the animated button to create CheckList, You can add title and list.

You can find four icons select the required icon:

  • Save - Saves in Notes
  • Reminder -Saves in Notes and Reminders
  • Favourite - Saves in Notes and Favourites
  • Cancel

Notes Page

In this page, you can view all your saved notes. This notes helps you in memorizing daily activities or the important information.

You can remove the completed tasks or items in checklist by clicking on checkbox it will appear in checkedlist. If you want to undo from checkedlist simply click on the checkbox

Any changes made to the list will reflect in favourites and reminders, if same list is there in favourites and reminders

You can find four icons when you place your cursor on your note:

  • Reminder - add to Reminders
  • Favourite - add to Favourites
  • Archive - Removes from Notes and add to Archives
  • Trash - add to Trash

Reminder Page

You can remove the completed tasks or items in checklist by clicking on checkbox it will appear in checkedlist. If you want to undo from checkedlist simply click on the checkbox

Any changes made to the list will reflect in Notes and Favourites, if same list is there in Notes and Favourites
